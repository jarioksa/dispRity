9 Future directions

9.1 More tests!

Some more tests are being developed such as a sequential.test to run sequential linear models for testing hypothesis in disparity through time or a model.test developed in collaboration with Mark Puttick to fit modes of evolution to disparity curves. Stay tuned!

9.2 More rarefactions!

I am working on including several “clever” bootstrap and rarefaction algorithms for palaeobiology studies based on geological sampling (following:and Butler et al. 2012 Evolution and Kotric and Knoll 2015 Palaeobiology).

9.3 Faster disparity calculations

I am slowly implementing parallel disparity calculation as well as C implementations of some disparity metrics to increase significantly improve the speed of the dispRity function.

9.4 More modularity

I am equally slowly developing functions to allow more of the options in the package to be modular (in the same way as the metric argument in dispRity). The next arguments to benefit this increased modularity will be chrono.subsets’s model argument and boot.matrix’s type argument.

9.5 More suggestions?

More generally, I welcome any suggestions or collaborations for developing some additional specific ideas. Feel free to contact me via email (guillert@tcd.ie) or if you are a GitHub user, directly create an issue on the GitHub page.
